# approx-grover-rudolph
Approximate generalization of the Grover-Rudolph algorithm

## Files Overview 
The code for the algorithm is in the folder `src/grover_rudolph`.
- `helping_functions.py`
- `approx_algorithm.py`
- `grover_rudolph.py`
- `gate_count.py`

In the `scripts/` folder, there are two files: 
- `simulation.py`
- `vector_simulation.py`

## Installation
If you want to edit the project, first install our grover rudolph package:
```
python -m pip install -e .
```
